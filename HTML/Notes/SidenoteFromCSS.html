We can use Id and class attribute for tags In vs code, Writing with Shortcuts are Easy For Example(using div) 
        For ID      => div#idname 
        For Class   => div.classname 
        For Writing two or more class 
            => div.class1.class2 

        We can use any tags other than div too
        By default it is div tags
        Example:
            => #id10    = 

            => .class10 =
