CSS Selectors

A CSS selector is used to select an HTML element(s) for styling
For Example : 

    body{
        background-color:blue;
    }

Element selector

It is used to select an element based on the tagname
For Example: 

    h1{
        color:red;
    }

ID selector

It is used to select an element with a given id
For Example: 

    #id01{
        background-color:grey;
    }

Class selector

It is used to select an element with a given class
For Example: 

    .class01{
        color:white;    
    }


Important Notes

We can group Selectors like this:
h1,h2,h3,div{
    colour: blue;
}
We can use element class selector like this:
p.red{
    colour: red;
}
A * can be used as a Universal Selector to select all the Elements
*{
    margin:0;
    padding:0;
}
An inline Style will override external and internal Styles

CSS Comments

/*This is a CSS comment*/
